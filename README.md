# 读迹 (NovelLog) v1.0

**读迹** 是一款专为重度小说读者设计的极简、离线优先的阅读记录工具。

它帮助你系统化地管理阅读历史，记录书名、主角、评分、自定义标签以及详细的阅读笔记（支持图片）。所有数据完全存储在本地（IndexedDB），无服务器交互，隐私安全且无需网络。

## ✨ 核心功能

### 1. 书籍管理
*   **多维度信息**：记录书名、主角（支持多个）、评分 (0-100) 和自定义标签。
*   **防重复检测**：输入书名时自动检测库中是否已存在，防止重复建档。

### 2. 阅读笔记 (时间轴)
*   **无限追加**：每本书支持添加多条笔记，形成阅读时间轴。
*   **富内容**：支持纯文本记录读后感、雷点或剧情摘要，并支持插入**图片**（存储在本地数据库中）。

### 3. 强大的检索与筛选
*   **全局搜索**：实时搜索书名、主角名、标签甚至笔记正文内容。
*   **智能排序**：支持按“最近更新”、“创建时间”或“评分”排序。
*   **标签筛选**：点击任意标签即可快速筛选同类书籍。

### 4. 数据安全与迁移
*   **完全离线**：使用浏览器原生 **IndexedDB** 存储，支持存储大量文本和图片数据。
*   **JSON 导入导出**：支持将所有数据导出为 JSON 文件进行备份或迁移设备。
*   **冲突处理**：导入数据时若遇到同名书籍，提供可视化的对比界面，支持“跳过”或“覆盖”。

### 5. 界面体验
*   **双色主题**：内置深色模式（Dark Mode）和浅色模式，跟随系统或手动切换。
*   **多语言**：支持简体中文和 English 实时切换。
*   **响应式设计**：完美适配手机端和桌面端操作。
